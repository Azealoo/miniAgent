# Western Blot 10x Running Buffer Scaling Protocol

## Purpose
Documentation of a real-world buffer scaling scenario where glycine supply was limited, requiring proportional adjustment of all components.

## Original Recipe (500mL 10x Running Buffer)
- Glycine: 72 g
- Tris base (Trizma): 15 g
- SDS: 5 g
- Bring to 500 mL with dH₂O

## Scaling Principles
1. All components must be scaled proportionally to maintain buffer chemistry
2. The limiting reagent determines the maximum possible volume
3. Practical considerations (round numbers, reagent conservation) should guide final choice

## Calculation Method
Scaling factor = (available limiting reagent) / (original amount of limiting reagent)

For glycine-limited scaling:
- Scaling factor = Available glycine (g) / 72 g
- New volume = 500 mL × scaling factor
- Other components = Original amount × scaling factor

## Example Scenarios

### Scenario 1: 30g glycine maximum
**Maximum volume approach:**
- Scaling factor = 30g / 72g = 0.4167
- New volume = 500 mL × 0.4167 = 208.3 mL
- Tris base = 15g × 0.4167 = 6.25 g
- SDS = 5g × 0.4167 = 2.08 g

**Practical approach (200 mL):**
- Scaling factor = 200 mL / 500 mL = 0.4
- Glycine = 72g × 0.4 = 28.8 g
- Tris base = 15g × 0.4 = 6.0 g
- SDS = 5g × 0.4 = 2.0 g

### Scenario 2: 50g glycine available
- Scaling factor = 50g / 72g = 0.6944
- New volume = 347.2 mL
- Tris base = 10.42 g
- SDS = 3.47 g

## Preparation Instructions
1. Weigh all components accurately
2. Add to appropriate container
3. Add approximately 80% of final volume dH₂O
4. Stir or mix until completely dissolved
5. Bring to final volume with dH₂O
6. Label with: "10x Running Buffer", date, and concentration

## Usage Notes
- Dilute 1:10 with dH₂O to make 1x running buffer for gel electrophoresis
- Store at room temperature
- Stable for several months if protected from contamination
